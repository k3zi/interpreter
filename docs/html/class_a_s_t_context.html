void ASTContext::Reference ( Id *  I)

A method called for every reference (r-value) to an Id in a CORE statement.

This method will ensure the Id is in the symbol table and has been initialized. This should not be called when an Id is being declared.

Parameters
Ian Id that has been referenced in the Parser.
Exceptions
std::stringif the Id is not in the symbol table.
